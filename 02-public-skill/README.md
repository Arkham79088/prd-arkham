# PRD Writer Skill（通用版）

适合大众使用的 AI PRD 写作 Skill，无需任何项目配置，开箱即用。

## 功能

- 自动拆解需求、提问不确定事项，确认后再写 PRD
- 按标准结构输出 PRD（背景、目标、功能详述、交互规范、异常处理）
- 自动生成 HTML 交互原型（含空状态、错误状态、加载状态）
- 支持口头描述、Word 文档、设计截图等多种输入方式

## 安装方式

### Claude Cowork / claude.ai

下载打包文件后点击「Copy to your skills」安装：

👉 [下载 prd-writer.skill](./prd-writer.skill)

### Cursor

将 `SKILL.md` 内容复制到项目的 `.cursorrules` 文件中。
