# PRD Writer · 标准版

开箱即用的 AI PRD 写作助手，无需任何配置。

## 功能

- 自动拆解需求、提问不确定事项，确认后再写 PRD
- 按标准结构输出 PRD（背景、目标、功能详述、交互规范、异常处理）
- 自动生成 HTML 交互原型，包含空状态、错误状态、骨架屏
- 支持口头描述、Word 文档、设计截图等多种输入方式

## 安装

### Claude Cowork / claude.ai
下载 [`prd-writer.skill`](./prd-writer.skill)，点击「Copy to your skills」安装，重启对话即可使用。

### Cursor
下载 [`prd-writer.mdc`](./prd-writer.mdc)，放入项目根目录的 `.cursor/rules/` 文件夹下即可生效。

## 适合谁

适合个人 PM、小团队，或第一次尝试 AI 写需求的产品经理。不需要配置项目结构，描述需求即可开始。
