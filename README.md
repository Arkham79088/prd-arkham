# PRD Arkham

面向产品经理的 AI 需求写作工具，提供标准版和专业版按需选用。支持 Claude Cowork 和 Cursor 两种使用方式。

---

## 选择适合你的版本

### 📝 标准版
适合个人 PM 和小团队，无需任何配置，描述需求即可开始。

👉 [查看安装说明](./标准PRD文档/README.md)

### ⚙️ 专业版
适合多系统/多市场团队，提供完整的项目层级、版本管理和 APP 专项支持。

👉 [查看安装说明](./专业PRD文档/README.md)
