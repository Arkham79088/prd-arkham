# PRD Arkham

一套面向产品经理的 AI 需求管理工具包，从轻量模板到完整 AI Skill，按需选用。

---

## 三个版本，按需选择

### 📄 01 · 通用 PRD 模板
**不需要 AI，直接填写**

适合：任何 PM，无工具依赖，团队统一格式
内容：标准 PRD 结构（背景、目标、功能详述、交互规范、异常处理）

👉 [查看模板](./01-universal-template/PRD-template.md)

---

### 🤖 02 · AI Skill 通用版
**开箱即用，无需配置**

适合：个人 PM、小团队，第一次尝试 AI 写需求
功能：需求拆解确认 → PRD 编写 → HTML 原型自动生成

👉 [查看说明 + 下载安装](./02-public-skill/README.md)

---

### ⚙️ 03 · AI Skill 专业版
**完整需求管理体系**

适合：多系统/多市场团队，需要版本管理和长期维护
功能：项目层级 · 版本管理 · 增量 PRD · APP 专项 · 索引维护

👉 [查看说明 + 下载安装](./03-pro-skill/README.md)
