# PRD Arkham · 专业版

适合有一定 AI 使用经验、需要完整需求管理体系的产品团队。

## 与标准版的区别

| 功能 | 标准版 | 专业版 |
|------|:------:|:------:|
| 需求拆解确认 | ✅ | ✅ |
| PRD + HTML 原型生成 | ✅ | ✅ |
| 微交互覆盖清单 | ✅ | ✅ |
| 原型全状态渲染 | ✅ | ✅ |
| 项目层级管理（市场/系统/模块） | ❌ | ✅ |
| 需求索引与版本管理 | ❌ | ✅ |
| 增量 PRD（v2.0 只写差异） | ❌ | ✅ |
| APP 专项（埋点/灰度/分流/BI） | ❌ | ✅ |
| 交叉引用与精度地图 | ❌ | ✅ |
| 首次配置 + 启动检测 | ❌ | ✅ |

## 安装

### Claude Cowork / claude.ai
下载 [`prd-arkham.skill`](./prd-arkham.skill)，点击「Copy to your skills」安装，重启对话即可使用。

### Cursor
下载 [`prd-arkham.mdc`](./prd-arkham.mdc)，放入项目根目录的 `.cursor/rules/` 文件夹下即可生效。`.mdc` 文件已完整内嵌所有模板和原型指南，无需额外文件。

## 适合谁

适合多系统、多市场并行的产品团队，或需要长期维护需求版本历史、管理 APP 专项文档的 PM。
